Se connecter

La page de connexion est la page que l'utilisateur se sert pour accéder au système. C'est la seul page qui est visible en accès direct et qui peut être accédée.

Perte de mot de passe

Si vous perdez votre mot de passe vous n'avez qu'à cliquer sur la phrase « Mot de passe oublié? ». Cette option vous enverra un mot de passe à l'adresse que vous avez spécifié lors de la création de votre compte.
