Logs

La page "Logs" est la page que l'utilisateur (Administrateur seulement) se sert pour accéder à la liste des actions ayant été effectuées par tous les utilisateurs.

Liste des logs

La liste des actions ayant été effectués par les utilisateurs du système :

  • Utilisateur: le nom de l'utilisateur ayant effectué une actions
  • Date: Date et heure à laquelle l'action à été fait.
  • Action sur: le type d'information ayant été modifier
  • Action: Le nom de l'action ayant été effectué (insertion, modification ou suppression d'informations.)

La ligne de navigation (ci-dessus) permet de choisir le nombre d'entrées (lignes) à afficher, permet de savoir combien de pages d'entrées y a t-il et permet de naviguer dans les pages (flèches rouges).

Actions sur la liste d'attente

Les actions possibles sur la liste d'attente sont (selon les droits utilisateurs):

  • Voir un Log

Voir

Lors d'un clic sur le bouton voir (ci-dessus), une fiche décrivant les divers changement apporté par l'action est affiché.
