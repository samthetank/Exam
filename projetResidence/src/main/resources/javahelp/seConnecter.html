Se connecter

Pour se connecter il suffit d'entrer son nom d'utilisateur ainsi que son mot de passe. Pour obtenir un mot de passe il faut contacter l'administrateur du système.

Perte de mot de passe

Si vous perdez votre mot de passe vous n'avez qu'à cliquer sur la phrase « Mot de passe oublié? ». Cette option vous enverra un mot de passe à l'adresse que vous avez spécifiée lors de la création de votre compte.
